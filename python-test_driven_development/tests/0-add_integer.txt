Add two integers:
>>> add_integer(1, 2)
3

Add integer and float:
>>> add_integer(100.3, -2)
98

Add with default value for b:
>>> add_integer(2)
100

Handle invalid inputs (string):
>>> try:
...     add_integer(4, "School")
... except Exception as e:
...     print(e)
...
b must be an integer

Handle invalid inputs (None):
>>> try:
...     add_integer(None)
... except Exception as e:
...     print(e)
...
a must be an integer
